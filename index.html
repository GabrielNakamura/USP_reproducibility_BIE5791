Boas Práticas e Ferramentas da Ciência Aberta na Ecologia - BIE5798-1

Apresentação

Olá!

Obrigado pelo interesse em participar da disciplina “Boas Práticas e Ferramentas da Ciência Aberta na Ecologia - BIE5798-1”, que será ministrada entre os dias 07 e 11 de Agosto na USP, São Paulo. Nesta disciplina serão abordados conceitos teóricos e ferramentas práticas para desenvolver pesquisas que vão ao encontro dos princípios de ciência aberta.

Este sítio servirá como um guia para nossas aulas. Nele vocês encontrarão os materiais necessários (dados) bem como os scripts que utilizaremos durante as aulas para as atividades práticas. Ah! O site continuará funcionando após o término da disciplina, portanto, sempre que tiver alguma dúvida é só voltar aqui e revisitar o que vimos nas aulas.

Dados

Com o intuito de manter o foco na compreensão dos conceitos e processos que serão ensinados, preferimos utilizar um material único compartilhado com toda turma. Para tanto vamos utilizar um conjunto de dados chamado Palmer Penguins, e em apenas algumas situações utilizaremos um outro conjunto de dados para demonstrar a funcionalidade de outras ferramentas (targets e docker).

O Palmer Penguins é um conjunto de dados com informações sobre três espécies de pinguins (ilustrados na imagem abaixo). Estes dados foram elaborados com o intuito de ensinar visualização e exploração de dados e funciona como uma alternativa a famosa base de dados iris.

Ao invés de utilizar diretamente o pacote palmerpenguins, pedimos que façam o download de uma tabela de dados extraída do pacote. Optamos por essa opção pois possibilita explorar algumas funcionalidades dentro do contexto de controle de versão e reprodutibilidade.
